# Life Logger Journal

A smart personal memory journaling Android app with:
- Voice journaling
- AI assistant chat and voice replies
- Photo memory storage
- Date, time, and location tagging
- Secure lock screen protection
- Beautiful modern UI with themes

## Features
- Save text, voice, and photo memories
- AI friend assistant
- Memory gallery
- Dark and light themes
- Custom color themes
- Secure PIN lock

## Built With
- Java
- Android Studio
- SQLite Database
- Text-to-Speech
- Speech Recognition
- Location Services
